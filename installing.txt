1. Download VSCode and install it - https://code.visualstudio.com/
2. Download python 3.8.7 and install it - remember to add to PATH and to disable path length - https://www.python.org/downloads/release/python-387/
3. Download python 3.8 from Windows Store
3. Download git - https://gitforwindows.org/
4. Open Git CMD and run C:\Users\[username] git clone https://github.com/Sam-Nielsen-Dot/HaileyburyExtracurricularWebsite
5. Open C:\Users\[username]\HaileyburyExtracurricularWebsite in VSCode
6. Select git bash as default terminal
7. Run git checkout -b "[name]sBranch"
8. Return to CMD and run cd ExtracurricularWebsite
9. Run python manage.py runserver
